# RMK_V2 - Spletna stran za taborniški rod

Projekt za prakso pri podjetju. Spletna stran je zgrajena z modernim tech stackom, ki omogoča hitrost in odzivnost.

## 🛠 Tech Stack
- **Framework:** Next.js (App Router)
- **Jezik:** TypeScript
- **Stiliranje:** TailwindCSS (brez zunanjih .css datotek)
- **Komponente:** ShadCN UI
- **Formatiranje:** Prettier
